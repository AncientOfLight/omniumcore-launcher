OmniumCore — Sounds
===================

Place 16-bit PCM .wav files here. The app loads:

- click.wav         → played when a game card is clicked
- launch.wav        → played when a game is launched
- notification.wav  → played when an update is available

The placeholder .wav files in this folder are tiny silent stubs
so the app starts cleanly. Just overwrite them with your own audio
keeping the same filenames — no code changes needed.
